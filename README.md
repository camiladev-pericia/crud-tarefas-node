# ✅ Task Manager CRUD
API para gerenciamento de tarefas e produtividade, focada em performance e simplicidade.

| Método | Rota | Descrição | Acesso |
| :--- | :--- | :--- | :---: |
| **GET** | `/tarefas` | Retorna todas as tarefas cadastradas | 🔓 Público |
| **POST** | `/tarefas` | Cria uma nova tarefa | 🔐 Privado |
| **PUT** | `/tarefas/:id` | Edita o status ou conteúdo da tarefa | 🔐 Privado |
| **DELETE** | `/tarefas/:id` | Remove uma tarefa do banco de dados | 🔐 Privado |
